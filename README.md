# ⚔️ Clint City Rivals

A fan-made, offline **Urban Rivals** clone — built with plain HTML/CSS/JavaScript, **no build step and no framework**. It uses **real Urban Rivals card data** (≈2,800 characters across all 36 clans) and renders loyal UR-style cards over Clint City backgrounds.

> ⚠️ **Fan project — not affiliated with Urban Rivals or Acute Games.** Card data, character art, clan emblems and the "Urban Rivals" font are property of their respective owners and are sourced from [iClintz](https://iclintz.com) and the [Urban Rivals Fandom wiki](https://urban-rivals.fandom.com). This repository is for personal, educational use only.

## Features

- **Real card data** — every character from all **36 clans** (~2,818 cards) with real names, stats, abilities and clan bonuses, scraped from iClintz.
- **Loyal card rendering** — real character art (UR CDN) composited under the authentic rarity frames (Common/Uncommon/Rare/Legendary), stat icons, level/XP gauge and the Urban Rivals webfont.
- **Authentic combat** — `Attack = Power × (1 + Pillz)`, highest attack wins, Fury, clan bonuses (≥2 of a clan), and **~98% of all card abilities faithfully simulated** (Copy, Cancel-modifier, Protection, Equalizer, Stop, Backlash, Growth/Degrowth, Support, Defeat/Killshot, poison/heal/mindwipe/dope, …). Effects that aren't simulated are shown but greyed out, so the UI never lies about what's active.
- **Urban-Rivals-style battle UI** — two rows of large cards that clash in place, real Clint City fight backgrounds, life/pillz HUD, player level, and per-card XP.
- **Deck builder & collection** — build a deck of 8, search/filter by clan, earn XP and level your cards up.

## Run it

It must be served over **HTTP** (not opened as a `file://`), because it fetches `data/cards-data.json` and loads remote character art.

```bash
# from the project root
python -m http.server 8137
# then open http://localhost:8137/index.html
```

(Any static server works — e.g. `npx serve`.)

## Refreshing the data / assets

Nothing is hard-coded; the dataset and look-and-feel assets are generated by scripts:

```bash
node tools/fetch-cards.js     # → data/cards-data.json  (scrapes iClintz, all clans)
node tools/fetch-assets.js    # → ui/assets/  (card frames, stat icons, UR font, Clint City backgrounds)
```

## Project layout

```
index.html            entry point (loads scripts with ?v= cache-busting)
data/
  cards.js            async loader: fetches cards-data.json, builds CLANS/CARDS + helpers
  cards-data.json     generated real card data
engine/
  abilities.js        parses ability/bonus text → structured effect model
  game.js             combat resolution engine
  portrait.js         renders a loyal UR card (frame + art + overlays)
ui/
  app.js              views: menu, deck builder, collection, battle, game over
  style.css           UR charcoal+gold theme
  assets/             frames, icons, font, backgrounds (from fetch-assets.js)
tools/
  fetch-cards.js      iClintz scraper
  fetch-assets.js     asset downloader
```

## Notes

- Character **art** is loaded on demand from the Urban Rivals CDN, so the collection needs a connection the first time cards scroll into view; frames/font/backgrounds are bundled locally.
- The remaining ~2% of abilities (e.g. `Versus:`, `Reanimate`/`Rebirth`, and a few one-off mechanics) are contextual and not meaningful in a single 4-round duel, so they're displayed but greyed out.
